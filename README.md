# The student academic portal system was developed to provide a simple, efficient, and reliable way of managing student's academic records in schools.    
# 🎓 Student Academic Portal System

A desktop-based **Student Academic Portal System** developed in **Python** using the **Tkinter GUI library**. The application allows administrators to manage student academic records, calculate grades automatically, store records in CSV files, and search student information through a simple and user-friendly interface.

---

##  Project Overview

The Student Academic Portal System is designed to simplify the management of student academic records for educational institutions. It provides an easy-to-use graphical interface where administrators can record student information, calculate grades, and securely store academic records.

This project demonstrates the use of **Python programming**, **Graphical User Interface (GUI) development**, and **file handling** using CSV files.

---

## Features

-  Secure Login System
- User Authentication
- Add Student Academic Records
-  Automatic Total and Average Calculation
-  Automatic Grade Assignment
-  Automatic Performance Comment
-  Search Student Records
- Display All Records
-  Add New System Users
- Clear Input Fields
-  Exit Confirmation
-  CSV File Storage
